# ☕ Estudos Java

Repositório dedicado aos meus estudos e exercícios práticos em Java, desenvolvido durante cursos, desafios e projetos pessoais.

O objetivo deste repositório é consolidar conhecimentos fundamentais da linguagem, boas práticas de desenvolvimento e conceitos utilizados no desenvolvimento backend.

## 📚 Conteúdos Estudados

### 🌐 Consumo de APIs

* Requisições HTTP
* Integração com APIs REST
* Manipulação de respostas JSON
* Uso de HttpClient

### 🔄 Conversão de JSON

* Serialização e desserialização de objetos
* Manipulação de JSON com bibliotecas Java
* Mapeamento de dados

### 🎯 Estruturas de Controle

* Condicionais (`if`, `else`, `switch`)
* Laços de repetição (`for`, `while`)
* Controle de fluxo

### 📋 Estruturas de Dados e Listas

* Collections Framework
* List e ArrayList
* Ordenação e filtragem
* Manipulação de coleções

### 📁 Manipulação de Arquivos

* Leitura de arquivos
* Escrita de arquivos
* Tratamento de dados persistidos

### 🧩 Polimorfismo e POO

* Herança
* Polimorfismo
* Encapsulamento
* Abstração
* Interfaces

### 📝 Sobrescrita do método toString()

* Representação textual de objetos
* Boas práticas de modelagem

### 💻 Exercícios Diversos

* Resolução de desafios de lógica
* Exercícios de fixação
* Prática dos conceitos estudados

## 🛠️ Tecnologias Utilizadas

* Java
* IntelliJ IDEA
* Git

Desenvolver uma base sólida em Java através da prática constante, explorando desde conceitos fundamentais até recursos utilizados em aplicações reais.
